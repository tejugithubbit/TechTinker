*****SOLAR SYSTEM USING OPENGL*****

Overview:
This project simulates the working of an actual Solar System.This project provides the following functionalities:
1.We can change the axis of rotation and colour of each planet to visualize the beauty of the Solar System.
2.We can set the delay timingf for the rotation of eacn planet.

Tools and Technologies:
Tool-Opengl
Programming language C++
IDE-Codeblocks

Concepts used 
1.Transformation Functions-Translation,Rotation,Scalling

Step for Implementation:
1.Download the Source code.
2.Run the source code in IDE to visualize the output.
