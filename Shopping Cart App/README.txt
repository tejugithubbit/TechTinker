*****Shopping Cart Using Java*****

Overview :
This project aims to create an attractive E-Commerce application. The features of this application includes :

1. Admin Interface :
i. Login
ii. Add/Delete/Update the Products  
iii. View the Customers
iv. Monitor the Product Status

2. Customer Interface :
1. Sign Up/Sign In
2. Add/Remove the products from the cart 
3. Buy the Products
4. Make Payment

Tools and Technologies :
Programming Language - Java
IDE - NetBeans
Database - MySQL

Steps for Implementation :

1. Download the source code
2. Run the Java code in NetBeans
3. Run the database code in MySQL
4. View the output.
