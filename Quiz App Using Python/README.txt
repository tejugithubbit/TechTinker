*****Quiz App Using Python*****

Overview:

This project simulates the working of an actual Quiz App.This project provides the following functionalities:
1.This App displays 10 questions to the players.
2.Each correct guess gets one point.
3.The player can retake the quiz if he wishes.

Tools and Technologies:

Programming language Python
IDE-Pycharm

Step for Implementation:
1.Download the Source code.
2.Run the source code in IDE to visualize the output.
