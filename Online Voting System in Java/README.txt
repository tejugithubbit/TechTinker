*****Online Voting System in Java*****

Overview :
This project serves to check the eligibilty of the candidates in an Election. This follows the general rules of eligibility :
1. If a candidate is elder than 18 years, he/she is eligible to vote.
2. Otherwise, a candidate is not eligible to vote.

Tools and Technologies :
1. Programming Language - Java
2. IDE - Eclipse
3. Object Oriented Programming Concepts

Steps for Implementation :
1. Download the source code of the project.
2. Run the code in Eclipse IDE to view the output.
